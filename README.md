## Descripción

Este repositorio contiene la solución de dos ejercicios propuestos en HackerRank como parte del parcial práctico de la asignatura Análisis y Diseño de Algoritmos.

Los ejercicios desarrollados fueron:

* Sparse Arrays (Strings)
* Intro to Tutorial Challenges (Search)

## Lenguaje utilizado

Java 8

## Cómo ejecutar

1. Abrir HackerRank.
2. Seleccionar Java 8 como lenguaje.
3. Copiar el contenido del archivo `Solution.java`.
4. Ejecutar el programa con los casos de prueba proporcionados por la plataforma.

## Algoritmos utilizados

### Sparse Arrays

Se utilizó un `HashMap` para almacenar la frecuencia de aparición de cada cadena de texto y posteriormente responder las consultas de manera eficiente.

### Intro to Tutorial Challenges

Se implementó el algoritmo de búsqueda binaria (Binary Search), aprovechando que el arreglo se encuentra ordenado para reducir el número de comparaciones realizadas durante la búsqueda.
